## Instructions for Adding Images

1. Download royalty-free images from Unsplash, Pexels, or Wikimedia Commons
2. Place images in `/public/images/` directory
3. Use descriptive filenames (e.g., `temple-bells.jpg`, `lotus-macro.jpg`)
4. Update this file with proper attribution for each image
5. Ensure images are optimized (use Next.js Image component)

## Image Guidelines

- **Portrait**: Professional, respectful portrait photo
- **Temple Images**: Stone architecture, temple exteriors/interiors, non-identifiable shrine elements
- **Gallery**: Diyas, lotus flowers, bells, stone carvings, abstract temple elements
- **Avoid**: Deity caricatures, identifiable deity images, kitsch representations
